# vim-config

1. `git clone --recursive https://github.com/xuelingkang/vim-config ~/.vim-config`
2. `mkdir ~/.vim`
3. `ln -s ~/.vim-config/pack ~/.vim/pack`
4. `ln -s ~/.vim-config/commonvimrc ~/.vimrc`
5. `ln -s ~/.vim-config/vimrc ~/.vimrc`
6. `ln -s ~/.vim-config/ideavimrc ~/.vimrc`
7. `mkdir ~/.vim/.backup ~/.vim/.swp ~/.vim/.undo`
